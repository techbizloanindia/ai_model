Leave Application Process

  1. Navigate to "Apply Leave" from main dashboard
  2. Select appropriate leave type from available categories
  3. Specify required dates and duration
  4. Submit application for managerial approval
  5. Monitor application status through the system

üìÖ Leave History Tracking

Expense Management Workflow

Four-Step Expense Process

Step 1: Create Expense

Create individual expenses, mileage claims, per diem allowances, and other eligible expenses

Step 2: Club Expenses

Group related expenses into a single claim for processing efficiency

Step 3: Submit for Approval

Submit the compiled claim to designated approvers for review

Step 4: Get Paid

Receive payment upon approval of the expense claim

Adding New Expenses

  • Use the "+ Add expense" button to create new entries
  • Ensure all required documentation is properly attached
  • Follow company policies for expense categories and limits
  • Maintain accurate records for audit purposes
