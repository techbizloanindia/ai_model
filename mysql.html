üíª Programming Language Examples

Python (PyMySQL)

import pymysql

# Connection configuration
config = {
    'host': 'localhost',
    'port': 3306,
    'user': 'your_username',
    'password': 'your_password',
    'database': 'your_database',
    'charset': 'utf8mb4'
}

# Create connection
try:
    connection = pymysql.connect(**config)
    print("‚úÖ Connected to MySQL successfully!")
    
    # Create cursor and execute query
    with connection.cursor() as cursor:
        cursor.execute("SELECT VERSION()")
        version = cursor.fetchone()
        print(f"MySQL version: {version[0]}")
        
except pymysql.Error as e:
    print(f"‚ùå Error connecting to MySQL: {e}")
finally:
    if connection:
        connection.close()

Node.js (mysql2)

const mysql = require('mysql2');

// Create connection
const connection = mysql.createConnection({
    host: 'localhost',
    port: 3306,
    user: 'your_username',
    password: 'your_password',
    database: 'your_database'
});

// Connect and test
connection.connect((err) => {
    if (err) {
        console.error('‚ùå Error connecting to MySQL:', err);
        return;
    }
    console.log('‚úÖ Connected to MySQL successfully!');
    
    // Execute query
    connection.query('SELECT VERSION()', (err, results) => {
        if (err) throw err;
        console.log('MySQL version:', results[0]['VERSION()']);
        connection.end();
    });
});

Java (JDBC)

import java.sql.*;

public class MySQLConnection {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/your_database";
        String username = "your_username";
        String password = "your_password";
        
        try {
            Connection connection = DriverManager.getConnection(url, username, password);
            System.out.println("‚úÖ Connected to MySQL successfully!");
            
            Statement statement = connection.createStatement();
            ResultSet resultSet = statement.executeQuery("SELECT VERSION()");
            
            if (resultSet.next()) {
                System.out.println("MySQL version: " + resultSet.getString(1));
            }
            
            connection.close();
        } catch (SQLException e) {
            System.err.println("‚ùå Error connecting to MySQL: " + e.getMessage());
        }
    }
}

üîç Troubleshooting Common Issues

‚ùå Access Denied Error

Error: Access denied for user 'username'@'localhost'

Solutions:

  • Verify username and password are correct
  • Check if user has necessary privileges
  • Ensure user can connect from the specified host
# Grant privileges to user
GRANT ALL PRIVILEGES ON database_name.* TO 'username'@'localhost';
FLUSH PRIVILEGES;

üö´ Connection Refused

Error: Can't connect to MySQL server on 'localhost'

Solutions:

  • Check if MySQL service is running
  • Verify the port number (default: 3306)
  • Check firewall settings
# Check MySQL service status
sudo systemctl status mysql

# Start MySQL service
sudo systemctl start mysql

üîê Security Best Practices

1 Use Strong Passwords

Create complex passwords with uppercase, lowercase, numbers, and special characters

2 Create Dedicated Users

Don't use root for application connections. Create specific users with limited privileges

CREATE USER 'app_user'@'localhost' IDENTIFIED BY 'strong_password';
GRANT SELECT, INSERT, UPDATE, DELETE ON app_database.* TO 'app_user'@'localhost';
3 Use SSL/TLS

Enable encrypted connections for production environments

4 Environment Variables

Store credentials in environment variables, not in code

# .env file
DB_HOST=localhost
DB_PORT=3306
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=your_database
